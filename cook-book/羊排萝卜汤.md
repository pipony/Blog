# 羊排萝卜汤

参考教程：https://www.xiachufang.com/recipe/102856374

好吃指数：⭐⭐⭐

难易指数：💦💦

<img src="http://cdn.huangxindi.com/img/yangpailuobotang.jpg" width="400px" align=center/>

### 准备

| 食材     | 调料 |
| -------- | ---- |
| 羊排块   | 葱   |
| 萝卜*0.5 | 盐   |
|          | 姜   |
|          | 料酒 |



---

### 步骤👩‍🍳

* 羊排冷水焯水

* 萝卜切滚刀块（⌛焯水时）

* 煮

  * 开水+羊排+葱段+姜+料酒，煮沸后转小火30分钟（珐琅锅）
  * 加萝卜再20分钟
  * 加葱花+盐，出锅
  
  


### PLUS😋

* 超简单！
* 下次水要多放点！
